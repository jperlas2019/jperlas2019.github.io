Outbreak: Source

Project information

  • Category: Web Game
  • Purpose: BCIT Agile Methodology Course
  • Project date: May 2020
  • Project URL: outbreaksource.herokuapp.com

An idle browser game, setting you as a bitcoin hacker using the Outbreak: Source OS. Hack as many firewalls as you can to gain bitcoins for your own benefit!

FEATURES:
  • • Automatic bitcoin gain, even offline!
  • • An item shop to empower your hacks
  • • A prestige system to reset your game for more power!
  • • A login system to save your progress
  • • A terminal to type in commands
