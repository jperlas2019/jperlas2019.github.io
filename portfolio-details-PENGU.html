PENGU - Discord Bot

Project information

  • Category: Python Application
  • Purpose: Personal Project
  • Project date: February 2022 - Present
  • Project URL: GitHub Link

A Discord Bot that allows users to play music in voice channels. Currently live on 6 Discors servers. Try out PENGU in your server! Check out the GitHub link.

FEATURES:
  • • Play music for everyone to listen to in the voice channel
  • • Supports Youtube links, Spotify tracks and public playlists, and searching!
  • • A queue system to queue up your favourite songs
  • • Skip unwanted songs
  • • Put a song first in queue to listen ASAP!
  • • Loop your favourite song to listen forever
